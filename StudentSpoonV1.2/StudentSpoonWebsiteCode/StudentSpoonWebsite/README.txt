STUDENT SPOON - UNIT 15 WEBSITE PROTOTYPE

How to run the site:
1. Keep all folders and files together.
2. Open index.html in a modern browser, or open the folder in Visual Studio Code and use Live Server.
3. The interactive prototype uses HTML, CSS and JavaScript only. It does not require a database or internet connection.

Main pages:
- index.html: homepage
- recipes.html: searchable/filterable recipe list
- recipe-details.html: full recipe, rating and comments
- submit-recipe.html: validated submission form
- budget-tips.html: budget advice and cost calculator
- about.html: purpose, limitations and accessibility statement

Prototype data note:
Recipe pricing is illustrative. Comments, ratings and submissions use localStorage, so they are saved only in the browser/device used for testing.
